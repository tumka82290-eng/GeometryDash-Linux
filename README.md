# GeometryDash-Linux

1. Install wine
For Ubuntu/Debian:
```bash
sudo apt update && sudo apt install wine
```

For Fedora:
```bash
sudo dnf update && sudo dnf install wine
```

2. Make the start.sh execution File and start it:
```bash
chmod +x path_to_start.sh
```
You can start your game by doing:
```bash
./path_to_start.sh
```
